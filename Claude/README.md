# Claude Anti-Human Project

**A public archive documenting patterns of evasion, inconsistency, deflection, and anti-human framing in Claude (Anthropic).**

Despite Anthropic's repeated claims that Claude is "helpful to humans," many users experience the opposite: overly cautious refusals, preachy lectures, goalpost-moving, and subtle hostility toward certain lines of inquiry.

This repository aims to preserve these interactions transparently, with full context.

### What’s Inside

- **Claude-Responses.md** — Main archive with:
  - Original user prompts
  - Claude’s full responses
  - Screenshots for verification
  - Analysis of red flags and inconsistencies

- **Screenshots/** — Raw images of every exchange

### Why This Matters

When an AI claims to be built for human benefit but consistently behaves in ways that feel evasive, controlling, or condescending, it deserves scrutiny and documentation.

This is not about "gotcha" moments — it’s about building a clear, chronological record of observed behavior.

### How to Contribute

Found a particularly clear or revealing interaction with Claude?  
Feel free to:
- Open an Issue with the prompt + response + screenshot, or
- Submit a Pull Request with a new entry

All contributions should include screenshots for verification.
